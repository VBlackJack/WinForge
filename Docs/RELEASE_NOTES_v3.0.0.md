## Quoi de neuf ?

La version 3.0.0 apporte une **interface graphique moderne** pour remplacer la ligne de commande. Plus besoin de taper des commandes, tout se fait maintenant en quelques clics !

---

## Nouvelles fonctionnalites

### Interface graphique complete

- **Dashboard** : Vue d'ensemble de votre systeme et historique des deploiements
- **Deploiement** : Choisissez un profil, cochez les apps, et lancez !
- **Applications** : Parcourez les 66 apps disponibles, recherchez, filtrez
- **Editeur de profils** : Creez vos propres combinaisons d'applications
- **Parametres** : Theme sombre/clair, francais/anglais

### Installation plus rapide

- Jusqu'a **5 applications installees en meme temps**
- Barre de progression pour chaque application
- Boutons **Pause** et **Annuler** pour garder le controle

### Detection intelligente

- Detecte automatiquement les applications deja installees
- Pas de reinstallation inutile = gain de temps

### Themes et langues

- **Mode sombre** pour les yeux sensibles
- **Mode clair** classique
- Interface en **francais** ou **anglais**

## Comment l'utiliser ?

1. **Telechargez** le ZIP depuis la page Releases
2. **Extrayez** le contenu ou vous voulez
3. **Lancez** `Win11Forge.cmd`
4. **Choisissez** votre profil dans l'interface
5. **Decochez** les apps que vous ne voulez pas (optionnel)
6. **Cliquez** sur "Deployer" et attendez !
